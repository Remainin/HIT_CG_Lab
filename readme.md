# 哈工大计算机学院-计算机图形学-实验

### 说明：

实现代码为C++，实现的框架为MFC框架，本文件下载后，通过打开其中的exe可执行文件直接可以使用本软件，也可以通过vs进行调试。<br>
软件打开后界面如下：<br>
![](https://github.com/Remainin/HIT_CG_Lab/blob/master/by1.png)
<br>
本实验绘制多边形的方法采用中点画线法，多边形填充算法采用了种子填充发和扫面线填充法，操作包括对多边形的各种旋转、对称、平移、错切等，矩形直线段裁剪采用的是编码法。<br>本软件同时包含一个简单动画，为移动的“文字和简单的图片”，如下图：<br>
![](https://github.com/Remainin/HIT_CG_Lab/blob/master/by2.png)
